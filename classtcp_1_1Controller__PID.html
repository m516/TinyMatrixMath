Detailed Description

template<tmm::Size num_inputs_and_outputs = 1, typename Scalar = float>
class tcp::Controller_PID< num_inputs_and_outputs, Scalar >

The classic discrete-time PID controller. It uses the system's time to compensate for function calls at irregular intervals (units are in seconds).

Template Parameters
Scalarthe common datatype to use for all operations.
num_inputs_and_outputsif not one, creates a vector of parallel PID controllers
